# 快速开始指南

欢迎使用学迹VidNotes！本指南将帮助你快速启动项目。

## 🚀 快速开始（5分钟）

### 1. 安装依赖
```bash
flutter pub get
```

### 2. 生成数据库代码
```bash
flutter pub run build_runner build --delete-conflicting-outputs
```

### 3. 下载字体（可选）
如果不下载字体，应用将使用系统默认字体。要使用自定义字体：

1. 访问 [Google Fonts](https://fonts.google.com/)
2. 下载 Noto Sans SC 和 Inter 字体
3. 将字体文件放到 `assets/fonts/` 目录
4. 重新运行应用

### 4. 运行应用
```bash
flutter run
```

就这么简单！🎉

## 📱 使用流程

### 典型使用场景

#### 场景1：导入和观看视频
1. 打开应用
2. 点击底部的"导入视频"按钮
3. 选择一个视频文件
4. 等待导入完成
5. 点击视频卡片开始播放
6. 使用播放器控制播放

#### 场景2：记笔记
1. 在视频播放页面
2. 当看到重要内容时暂停
3. 点击右下角的添加笔记按钮
4. 选择笔记类型（重点/笔记/问题）
5. 输入笔记内容
6. 设置重要程度和标签
7. 点击保存
8. 继续观看，重复以上步骤

#### 场景3：复习笔记
1. 进入"笔记"标签页
2. 查看所有笔记列表
3. 点击笔记卡片查看详情
4. 点击笔记可以跳转到视频的对应位置
5. 编辑或删除笔记

## 🔧 常见问题

### Q: 视频导入失败？
A: 检查以下几点：
- 视频格式是否支持（mp4、mov、avi等）
- 视频大小是否超过500MB
- 是否授予了存储权限

### Q: 播放器黑屏？
A: 可能的原因：
- 视频编码格式不支持
- 视频文件损坏
- 尝试重新导入视频

### Q: 笔记无法保存？
A: 确保：
- 至少输入了笔记内容或选择了字幕文本
- 数据库已正确初始化
- 查看日志查找具体错误

### Q: 字体显示异常？
A: 
- 下载并放置正确的字体文件到 `assets/fonts/`
- 运行 `flutter clean && flutter pub get`
- 重新启动应用

## 📊 数据管理

### 数据存储位置
所有数据存储在应用私有目录：
- **数据库**: `{AppDocuments}/vidnotes_db.isar`
- **视频文件**: `{AppDocuments}/videos/`
- **截图**: `{AppDocuments}/screenshots/`

### 备份数据
目前暂不支持自动备份。可以手动复制上述目录进行备份。

### 清空数据
在"设置" > "数据" > "清空所有数据"

⚠️ **警告**: 此操作不可恢复！
