## 📱 设计规范

### 圆角系统
```dart
小：8px   - 小型组件
中：12px  - ActionChip、输入框
大：16px  - Card、按钮
超大：20px - 特殊卡片（欢迎卡片）
```

### 间距系统
```dart
XS：4px   - 最小间距
S：8px    - 小间距
M：16px   - 标准间距
L：24px   - 大间距
XL：32px  - 超大间距
```

### 阴影系统
```dart
亮色模式：黑色 10% opacity
暗色模式：黑色 30% opacity
elevation：2-4 为主
```

### 颜色系统（暗色模式）
```dart
背景：#000000（纯黑）
卡片：#1C1C1E（深灰）
强调色：沿用 AppColors
文字：白色 90% opacity（主要）
文字：白色 60% opacity（次要）
```

## 🎓 设计原则（学习自 Deja）

1. **简约至上**：去除不必要的装饰
2. **层次分明**：使用阴影和间距区分内容
3. **一致性**：统一的圆角、间距、颜色
4. **易用性**：清晰的导航、明显的交互反馈
5. **现代感**：Material 3 + 自定义优化

---

## 📝 注意事项

1. **性能**：卡片阴影不要过多，影响性能
2. **适配**：确保在不同屏幕尺寸下表现良好
3. **对比度**：暗色模式下注意文字可读性
4. **动画**：过渡动画时长控制在 200-400ms
